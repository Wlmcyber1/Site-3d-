🚀 MDev - Email for Developers


Uma Landing Page moderna e responsiva desenvolvida para simular uma plataforma de serviços de e-mail voltada para desenvolvedores. O projeto foca em uma interface futurista ("Dark Mode") com elementos 3D e animações fluidas.


✨ Demonstração Visual
Paleta de Cores: Fundo Dark (#000000) com detalhes em tons de cinza e roxo.

Efeito de Vidro (Glassmorphism): Utilização de camadas de desfoque e gradientes.

Interatividade: Botões com efeitos de hover e animações de entrada.



🛠️ Tecnologias e Bibliotecas
HTML5 & CSS3: Estruturação e estilização avançada com Flexbox e Keyframes.

Spline: Integração de um modelo 3D interativo de robô diretamente no navegador.

AOS (Animate On Scroll): Biblioteca utilizada para as animações de surgimento dos elementos conforme o rolagem da página.

Google Fonts: Tipografia limpa e moderna integrada via system-ui.





Certifique-se de que o arquivo style.css e a imagem gradient.png (se disponível) estejam na mesma pasta do index.html.

Abra o arquivo index.html em qualquer navegador moderno.

Nota: É necessária conexão com a internet para carregar os scripts do Spline e do AOS via CDN.
